The Abjad tempo token

All Abjad components afford tempo assignment. 

abjad> note = Note(0, (1, 4))
abjad> note.tempo = ((3, 8), 52)

Abjad defines the tempo token equal to the pair of a duration token and any integer or float interpretable as a metronome marking.
