The core duration interface

The core duration interface implements three public attributes. All three are read-only.

  • prolation
  • prolations
  • prolated

See the chapters on prolation and prolated duration.

The core duration interface implements no public methods.

The core duration interface inherits from the Abjad Rational class. This means that all Abjad duration interfaces afford the standard arithmetic operations, like addition, multiplication, exponentiation and comparison.

Comments

The core duration interface currently inherits from both _Interface and Rational. Smarter will be to strip the multiple inheritence and leave the core duration interface to inherit from _Interface only. This will greatly reduce the number of arithmetic operations which the different duration interfaces can undergo; and this is a good thing. We want duration interfaces to afford comparison but not arithmetic. This means that we will have to implement comparison on the core duration interface by hand after we remove Rational from the inheritance chain.

The classname of the core duration interface is _DurationInterface.The class implements in the core/duration module.
