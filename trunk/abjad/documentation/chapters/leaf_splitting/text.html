Leaf Splitting

An Abjad _Leaf with duration d can be split in two Leaves with durations e and f such that e + f = d.

There are two helper functions for doing this: leaf_split( ) and leaf_split_binary( ).

Both functions take two parameters: the duration (n, d) at which the _Leaf will be broken, and the Leaf to break.

leaf_split( ) will accept any duration smaller than the duration of the _Leaf to split.
leaf_split_binary( ) will only take durations with a power of 2 as denominator.

Both functions behave the same way when resulting durations e and f are notehead-assignable (i.e. e ∈ (D1 ∪ D2) ),

leaf_split( ) leaf_split_binary( ) 
         abjad> n = Note(0, (1, 4))
         abjad> leaf_split((1, 8), n)
         [Note(c', 8), Note(c', 8)]
      
         abjad> n = Note(0, (1, 4))
         abjad> leaf_split_binary((1, 8), n)
         [[Note(c', 8)], [Note(c', 8)]]

Notice that leaf_split( ) returns a list of 2 Leaves, while leaf_split_binary( ) returns a list of two lists, each with a _Leaf in this case. The reason for this will become clear.

When the duration attribute is not notehead-assignable, leaf_split( ) will wrap the splitted Leaves in a Tuplet;
leaf_split_binary( ) will return splitted Leaves.

leaf_split( ) leaf_split_binary( ) 
         abjad> n = Note(0, (1, 4))
         abjad> leaf_split((5, 32), n)
         [FixedDurationTuplet(5/32, [c'4]), Note(c', 16.)]
      
         abjad> n = Note(0, (1, 4))
         abjad> leaf_split_binary((5, 32), n)
         [[Note(c', 8), Note(c', 32)], [Note(c', 16.)]]
