The Abjad _FMTupletDurationInterface class manages the duration attributes of fixed-multiplier tuplets.

Example 

abjad> tuplet = FixedMultiplierTuplet((5, 6), Note(0, (1, 16)) * 6)
abjad> Beam(tuplet)
abjad> outer = FixedMultiplierTuplet((2, 3), [Note(0, (1, 16)), tuplet])
abjad> measure = Measure((2, 8), [outer])
abjad> staff = RhythmicStaff([measure])
abjad> show(staff)
