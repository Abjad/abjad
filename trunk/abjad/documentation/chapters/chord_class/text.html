articulations

Read / write attribute inherited from _Leaf.

Return type

Returns an Abjad _ArticulationsInterface. 

abjad> chord = Chord([0, 2, 10], (1, 4))

abjad> chord.articulations
Articulations( )

Assignment values

Assign a list of articulation strings. 

abjad> chord = Chord([0, 2, 10], (1, 4))
abjad> chord.articulations = ['staccato', 'marcato']

Assign None to clear. 

abjad> chord = Chord([0, 2, 10], (1, 4))
abjad> chord.articulations = None

next

Read-only attribute inherited from _Leaf.

Return type

Returns a reference to next-leaf-in-container after chord. 

abjad> note1 = Note(-1, (1, 4))
abjad> chord = Chord([0, 2, 10], (1, 4))
abjad> note2 = Note(9, (1, 4))
abjad> measure = Measure((1, 4), [note1, chord, note2])

abjad> chord.next
Note(a', 4)

Assignment values

The next attribute is read-only.

noteheads

Read / write attribute inherited from _Leaf.

Return type

Returns a Python list of references to Abjad _NoteHead objects. 

abjad> chord = Chord([0, 2, 10], (1, 4))

abjad> chord.noteheads
[_NoteHead(c'), _NoteHead(d'), _NoteHead(bf')]

Assignment values

Assign a list of numbers or tuple of numbers. Abjad interprets lists and tuples of numbers as notehead pitches. 

abjad> chord = Chord([0, 2, 10], (1, 4))
abjad> chord.noteheads = [-1, 3, 9, 11]

number

Read-only attribute inherited from _Leaf.

Return type

Returns a zero-indexed integer representing the in-container position of chord. 

abjad> measure = Measure((4, 4), [Note(x, (1, 8)) for x in range(-1, 7)])
abjad> measure[6] = Chord([0, 2, 10], (1, 8))

abjad> chord.number
6

Assignment values

The number attribute is read-only.

offset

Return type

staff

Read / write attribute inherited from _Leaf.

Return type

Returns a reference to the Abjad Staff object that governs chord, else None if no staff governs chord. 

abjad> chord = Chord([0, 2, 10], (1, 4))

abjad> chord.staff is None
True

Assignment values

Assign a reference to an Abjad Staff object. 

abjad> chord = Chord([0, 2, 10], (1, 4))
abjad> bottom_staff = Staff([ ])
abjad> chord.staff = bottom_staff

Assign None to clear. 

abjad> chord = Chord([0, 2, 10], (1, 4))
abjad> bottom_staff = Staff([ ])
abjad> chord.staff = None
