noteheads

Read / write attribute defined on Chord.

Return type

Returns a Python list of references to Abjad _NoteHead objects. 

abjad> chord = Chord([0, 2, 10], (1, 4))

abjad> chord.noteheads
[_NoteHead(c'), _NoteHead(d'), _NoteHead(bf')]

Assignment values

Assign a list of numbers or tuple of numbers. Abjad interprets lists and tuples of numbers as notehead pitches. 

abjad> chord = Chord([0, 2, 10], (1, 4))
abjad> chord.noteheads = [-1, 3, 9, 11]

Method examples

append(arg)

Bound method defined on Chord.

Interpret arg as a notehead and append to chord.

Valid types for arg:

  • int
  • long
  • float
  • Abjad Pitch
  • Abjad _NoteHead

Example 

abjad> chord = Chord([0, 2, 10], (1, 4))

abjad> chord.append(9)

copy( )

Bound method inherited from _Component.

Returns a new copy of chord.

Accepts no input parameters.

See example.

extend(arglist)

Bound method defined on Chord.

Interpret arglist as a list of noteheads and extend chord.

arglist must be a Python list of any of the following:

  • int
  • long
  • float
  • Abjad Pitch
  • Abjad _NoteHead

Example 

abjad> chord = Chord([0, 2, 10], (1, 4))

abjad> chord.extend([9, 16, 18])

__contains__(arg)

Special method defined on Chord.

Return True if arg in chord, and False otherwise.

Valid types for arg:

  • int
  • long
  • float
  • Abjad Pitch
  • Abjad _NoteHead

Example 

abjad> chord = Chord([0, 2, 10], (1, 4))

Special method defined on Chord.

Where:

  • pitchtokenlist is a Python list of zero or more pitch tokens
  • durationtoken is a valid duration token
