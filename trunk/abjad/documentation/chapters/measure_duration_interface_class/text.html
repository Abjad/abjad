Examples 

abjad> container = Container(Note(0, (1, 8)) * 5)
abjad> tuplet = FixedDurationTuplet((5, 8), [container, Note(0, (1, 8))])
abjad> measure = Measure((5, 8), [tuplet])
abjad> staff = RhythmicStaff([measure])
abjad> measure
Measure(5/8, [{@ 6:5 (c'8, c'8, c'8, c'8, c'8), c'8 @}])
abjad> show(staff)

Comments

The nonbinary, multiplier and prolated attributes handle measures governed by nonbinary meter.

See the _MeasureDurationInterface class in measure/duration.py for implementation details.
