The Abjad Skip class

The Abjad Skip class wraps the LilyPond skip object. LilyPond skips represent unprocessed periods of musical time, somewhat like invisible rests.

Public interface

The Abjad Skip class defines the following public interface.

read-only
  • clef
  • format
  • leaves
  • number
  • offset
  • staff
read-writer
  • accidentals
interfaces
  • articulations
  • barline
  • beam
  • dots
  • duration
  • dynamics
  • formatter
  • glissando
  • grace
  • harmonic
  • spanners
  • staff
  • stem
  • tempo
  • tie
  • tremolo
  • trill
navigation
  • next
  • prev
methods
  • __mul__
  • __rmul__
  • copy( )
  • kind( )

Skip inherits this interface in its entirety from the abstract _Leaf class and implements no additional attributes of its own.

See the chapter on the Abjad _Leaf class for definitions of these attributes.
