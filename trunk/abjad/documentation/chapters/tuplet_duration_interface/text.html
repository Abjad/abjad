augmentation: Boolean. True when tuplet multiplier is greater than 1. False otherwise.

diminution: Boolean. True when tuplet multiplier is less than 1. False otherwise.

prolated: nonnegative rational. See prolated duration.

prolation: positive rational. See prolation.

prolations: list of positive rationals. See prolation.

Comments

This abstract tuplet duration interface probably should have a multiplier attribute. Both the concrete fixed-duration tuplet interface and the concrete fixed-multiplier tuplet interface have multiplier attributes. This abstract tuplet duration interface lacks a multiplier attribute because the two conrete children duration interfaces implement that attribute differently — in one case as a user-settable property and in the other case as a read-only property. Would be nice to figure out the python that will allow this abstract tuplet duration interface to have a multiplier value anyway.

See _TupletDurationInterface in tuplet/duration.py for the implementation.
