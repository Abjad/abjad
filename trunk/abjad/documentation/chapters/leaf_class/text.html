Examples

The Abjad _Leaf class is an abstract class and is not meant to be instantiated by hand.

But we show examples of _Leaf attributes here using note, rest and chords as concrete examples of the _Leaf class.

abjad> chord.articulations = ['staccato', 'marcato']

next

Read-only attribute defined here.

Return type

Returns a reference to next-leaf-in-container after chord. 

abjad> note1 = Note(-1, (1, 4))
abjad> chord = Chord([0, 2, 10], (1, 4))
abjad> note2 = Note(9, (1, 4))
abjad> measure = Measure((1, 4), [note1, chord, note2])

abjad> chord.next
Note(a', 4)

Assignment values

The next attribute is read-only.

number

Read-only attribute defined here.

Return type

Returns a zero-indexed integer representing the in-container position of chord. 

abjad> measure = Measure((4, 4), [Note(x, (1, 8)) for x in range(-1, 7)])
abjad> measure[6] = Chord([0, 2, 10], (1, 8))

abjad> chord.number
6

Assignment values

The number attribute is read-only.

Read-only attribute defined here.

abjad> measure[6] = Chord([0, 2, 10], (1, 8))

abjad> chord.offset

Assignment values

The offset attribute is read-only.

Override examples

__init__(args*)

Special method defined on Chord.

Initialize chord according to one of several input argument patterns:

  • Chord(pitchtokenlist, durationtoken)
  • Chord(note)
  • Chord(rest)
  • Chord(skip)

Where:

  • pitchtokenlist is a Python list of zero or more pitch tokens
  • durationtoken is a valid duration token

    • Example 

    abjad> chord = Chord([0, 2, 10], (1, 4))

    __mul__(n)

    Special method inherited from Chord.

    Implements left-handed multiplication against Chord.

    Returns a Python list of n copies of chord.

    Example 

    abjad> chord = Chord([0, 2, 10], (1, 4))

    abjad> chords = chord * 4
abjad> chords
[Chord(c' d' bf', 4), Chord(c' d' bf', 4), Chord(c' d' bf', 4), Chord(c' d' bf', 4)]
