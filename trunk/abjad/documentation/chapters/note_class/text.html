Special methods

__init__(args*)

Special method defined here.

Initialize self according to one of several input argument patterns:

  • Chord(pitch, dur)
  • Chord(rest)
  • Chord(chord)
  • Chord(skip)

Where pitch is an Abjad pitch token, and where dur is an Abjad duration token. 

abjad> note = Note(0, (1, 4))
