Usage examples

Notes, rests, chords and skips all initialize exactly one _LeafDurationInterface. 

   abjad> measure = Measure((4, 4), [ ])
   abjad> measure.append(FixedDurationTuplet((2, 4), Note(0, (1, 4)) * 3))
   abjad> measure.append(FixedDurationTuplet((2, 4), Note(0, (1, 4)) * 3))
   abjad> show(measure)

   abjad> note = measure.leaves[0]

multiplier 

   abjad> note.duration.multiplier is None
   True

   abjad> note.duration.multiplier = (1, 2)
   abjad> print(note.format)
   c'4 * 1/2

   abjad> note.duration.multiplier = None
   abjad> note.duration.multiplier is None
   True

prolated 

   abjad> note.duration.prolated
   Rational(1, 6)

prolation 

   abjad> note.duration.prolation
   Rational(2, 3)

prolation 

   abjad> note.duration.prolations
   [Rational(2, 3)]

written 

   abjad> note.duration.written
   Rational(1, 4)

   abjad> note.duration.written = (3, 8)
   abjad> note
   Note(c', 4.)
