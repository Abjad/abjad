The Abjad Note class

The Abjad Note class models musical notes. This chapter describes the different bound methods defined against the Abjad Note class.

Creating notes

The Abjad Note class initializes with numeric pitch and duration values. 

  abjad> note = Note(0, (1, 4))
  abjad> note
  Note(c', 4)

Getting information about notes

Abjad organizes note attributes into a collection of interfaces. The note interfaces list alphabetically below.

  • duration
  • pitch
  • notehead

Working with pitches

You can assign pitches to a note a note in a couple of different ways. 

  abjad> note.pitch = 1
  abjad> note
  Note(cs', 4)

  abjad> note.pitch = Pitch('d', 4)
  abjad> note
  Note(d', 4)

  abjad> note.pitch = ('ds', 4)
  abjad> note
  Note(ds', 4)

See the chapter on pitch for more information.

Comments

The current implementation of Abjad allows the creation of notes without pitch. For example Note(None, (1, 4)) and also note.pitch = None. Later implementations of Abjad may deprecate this behavior.
