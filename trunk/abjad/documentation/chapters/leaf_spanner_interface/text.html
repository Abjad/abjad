The Abajad _LeafSpannerInterface class instantiates automatically at leaf-initialization time.

The _LeafSpannerInterface gives you access to all the spanners associated with a particular _Leaf instance. Through this interface one can get spanners, kill (die( )) spanners, split (fracture( )) spanners, fuse spanners and perform queries.
